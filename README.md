### BMI 계산하기
  * 공식 = 몸무게(kg)/(키(m) * 키 (m))  
  
    * 18.5 이하 -> 저체중  
    * 18.5 ~ 23 -> 정상체중  
    * 23 ~ 25 -> 과체중
    * 25 ~ 30 -> 경도 비만
    * 30 ~ 35 -> 중도 비만
    * 35 이상 -> 고도 비만
